

# 📦 Big Event Interface (大事件后台管理系统 API)的后端

> 基于 Node.js (Express) + MySQL 开发的前后端分离项目服务端接口。
>
> 本项目是黑马程序员《大事件》项目的**代码复现与功能扩展**。

## 🛠 技术栈 (Tech Stack)

  * **核心框架**: `Node.js`, `Express`
  * **数据库**: `MySQL`
  * **身份认证**: `JWT (JSON Web Token)` / `express-jwt`
  * **加密**: `bcryptjs` (密码加密)
  * **参数校验**: `joi` / `@escook/express-joi`
  * **跨域处理**: `cors`

## ✨ 功能特性 (Features)

  * **用户管理**: 注册、登录、个人信息查询/修改、重置密码、更换头像。
  * **文章分类**: 获取、新增、删除、根据 ID 获取分类、更新分类。
  * **文章管理**: 发布文章（支持封面上传）、获取文章列表、删除文章等。
  * **统一处理**:
      * 全局错误中间件处理。
      * Token 身份认证与拦截。
      * 表单数据合法性验证。

## 🚀 快速开始 (Run Locally)

如果你想在本地运行该项目，请按照以下步骤操作：

**1. 克隆项目**

```bash
git clone https://github.com/你的用户名/你的仓库名.git
```

**2. 安装依赖**
进入项目目录并安装 NPM 依赖：

```bash
cd 你的仓库名
# 推荐使用 pnpm (如果你没装 pnpm，用 npm install 也可以)
pnpm install
```

**3. 配置数据库**

  * 请在本地 MySQL 中新建数据库 `my_db_01` (根据你实际的数据库名修改)。
  * 导入根目录下的 `db.sql` 文件以初始化表结构。（如果你有 SQL 文件的话）
  * 修改 `db/index.js` (或配置文件) 中的数据库连接信息（账号/密码）。

**4. 启动服务**

```bash
node app.js
# 或者使用 nodemon
npm run dev
```

服务默认运行在 `http://127.0.0.1:3007`
